## 1. 세계관 개요

### 종족 체계

| 종족 | 별칭 | 계열 | 상징 | 에메랄드 포인트 |
|------|------|------|------|----------------|
| Kenine | The Walkers | 개과 (Fox, Wolf, Dog…) | 행동 · 용기 · 성장 | 볼털 X |
| Fenine | The Seekers | 고양이과 (Cat, Lion, Lynx…) | 지혜 · 기억 · 통찰 | 귀 안 ∞ |
| Lutrine | The Makers | 수달계 (Otter, Beaver…) | 창조 · 발명 · 적응 | 꼬리 끝 ⚙ |
| **Shattered** | The Shattered | 위 3종족의 타락 버전 | 집중력/기억/목표의 파편화 | 균열된 시그니처 |

> Shattered는 "악"이 아니라 **길을 잃은 상태**. 플레이어가 구출하는 대상.

### 종족 문장 (Emblem)
- Kenine → 🧭 나침반 ("행동이 길을 만든다")
- Fenine → 📖 열린 책 ("지식과 기억")
- Lutrine → ⚙ 기어 + 물결 ("창조와 혁신")
- Shattered → 💎 금 간 수정 ("왜곡과 분열")

## 3. Pax — Kenine 대표

### 기본 정보
```
이름:   Pax
종족:   Kenine (Fox)
포지션: Tier 0 — Lexiox의 얼굴
역할:   모든 광고, 앱 아이콘, 온보딩의 중심
성격:   에너지 넘침, 긍정적, 일단 해보는 타입
```

### 핵심 원칙
- 똑똑함 ✔ / 친근함 ✔ / 성장 ✔
- 유아틱 ❌ / 너무 귀여움만 강조 ❌

### 시그니처: 볼털 X
```
평시:   "귀여운 여우 볼털"
아이콘: "어? X네?"
Shattered: X의 한쪽 라인이 어긋남
```

### 비율
| 단계 | 머리 | 몸 | 꼬리 |
|------|------|-----|------|
| Cub | 60% | 25% | 15% |
| Master | 45% | 40% | 15% |

### 눈
- 형태: 아몬드형
- 색상: 에메랄드 (#00C7A5)
- 특징: 홍채 안에 육각 크리스탈 패턴

### 성장 6단계

| 단계 | 특징 | 에메랄드 포인트 | 장비 |
|------|------|----------------|------|
| Lv1 Cub | 큰 눈, 작은 꼬리 | 꼬리 끝 작은 점 | 없음 |
| Lv2 Explorer | 선명한 X | 2곳 | 나침반 + 작은 가방 |
| Lv3 Scholar | X + 안경 | 2곳 | 안경 + 책/스크롤 |
| Lv4 Guardian | X 에메랄드 발광 | 4곳 | 에메랄드 망토 + 문장 |
| Lv5 Master | X 크리스탈화 | 전신 | 크리스탈 브로치 + 지팡이 |
| Shattered | X 균열/어긋남 | — | 없음 (채도 -40%) |

### Shattered Pax
```
눈:    에메랄드 → 회색 균열 (한쪽)
볼털:  X 한쪽 라인 어긋남
꼬리:  에메랄드 끝 → 깨진 파편
채도:  -40%
표정:  슬픔 (공포/분노 ❌)
설정:  성장하지 못한 Pax. "길을 잃은 모습."
```

### 표정 세트 8종
`Happy` `Thinking` `Celebrate` `Sleepy` `Surprise` `Confident` `Study` `Streak`

## 4. Lyra — Fenine 대표

### 기본 정보
```
이름:   Lyra
종족:   Fenine (Cat)
포지션: Tier 1 — 첫 번째 동료
역할:   독해 · 문법 · 기억 · 통찰
성격:   조용하지만 깊음. 말수 적지만 한마디가 정확함.
```

### 시그니처: 귀 안 ∞
```
평시:   "귀 안쪽 자연스러운 털 무늬"
아이콘: "어? ∞ 네?"
Shattered: ∞ 한쪽이 끊어진 형태
```

### 실루엣 차별화
```
귀:  뾰족하고 길다 (Pax보다 15% 길게), 안쪽 보라빛 포인트
꼬리: 길고 가늘며 끝이 살짝 말림
몸:  날렵하고 길쭉한 비율
```

### 눈
- 형태: 아몬드형보다 더 가늘고 날카롭게
- 색상: 인디고/딥 퍼플 (#5B4FCF)
- 특징: 눈 안에 다면체 프리즘 패턴

### 색상
```
Primary:  인디고 #5B4FCF
Secondary: 딥 퍼플 #3D2FA0
Fur:      은회색 + 크림 베이지
에메랄드: 귀 끝 or 브로치 (공통 포인트)
```

### 성장 장비
`Cub: 없음` → `Explorer: 스크롤` → `Scholar: 가는 안경 + 책` → `Guardian: 인디고 망토` → `Master: 크리스탈 프리즘 펜던트`

### 표정 세트 8종
`Calm` `Analyzing` `Insight` `Focus` `Warm` `Determined` `Tired` `Proud`

> 표정 변화가 적고 섬세함. 크게 웃기보다 살짝 미소.

### Shattered Lyra
```
눈:    인디고 → 탁한 회보라, 프리즘 균열
귀 무늬: ∞ → 한쪽 끊어짐
꼬리:  말린 끝이 풀리고 축 처짐
설정:  기억이 파편화된 Lyra. "모든 걸 알았지만 잃어버린 존재."
```

```
귀:  작고 둥근형 (Pax/Lyra의 뾰족한 귀와 대비)
꼬리: 넓적하고 두꺼움 (수달 꼬리)
몸:  셋 중 가장 넓고 탄탄함
```

### 눈
- 형태: 크고 둥근형 (셋 중 가장 둥금)
- 색상: 황금/앰버 (#F5A623)
- 특징: 눈 안에 기어 톱니 패턴

### 색상
```
Primary:  앰버/골드 #F5A623
Secondary: 딥 앰버 #C4851A
Fur:      갈색 + 크림 베이지 (수달 투톤)

### 성장 장비
`Cub: 없음` → `Explorer: 렌치/메모패드` → `Scholar: 도면 + 두꺼운 안경` → `Guardian: 공구 벨트 + 앰버 망토` → `Master: 에메랄드 크리스탈 공구 + 홀로그램`

### 표정 세트 8종
`Excited` `Focused` `Eureka` `Oops` `Let's go` `Retry` `Overload` `Complete`

> 표정 전환이 빠름. 실패해도 0.5초 만에 Excited로 돌아옴.

### Shattered Rivet
```
눈:    앰버 → 탁한 녹슨 금색, 기어 패턴 멈춤
꼬리:  ⚙ 톱니 한두 개 빠짐
도구:  부러진 렌치 or 구겨진 도면
설정:  창조 의지를 잃은 Rivet. "아이디어가 떠오르지 않는 상태."

## 7. 온보딩 퀴즈 UX

### 흐름
```
앱 첫 실행 → Pax 인트로 → 질문 3개 → 종족 배정 → 홈 진입
```

### Pax 인트로
```
"안녕! 나는 Pax야. 🦊
Lexiox에 온 걸 환영해.
근데 있잖아, 사람마다 배우는 방식이 달라.
딱 3가지만 물어볼게. 솔직하게 대답해줘."
```

### 질문 3개

**Q1. 새로운 걸 배울 때 나는?**
- 🦊 일단 해보고 틀리면서 익힌다
- 🐱 이해가 될 때까지 먼저 생각한다
- 🦦 직접 뭔가 만들거나 써보면서 익힌다

**Q2. 어려운 문제를 만나면?**
- 🦊 일단 도전. 틀려도 괜찮아
- 🐱 왜 어려운지 먼저 파악한다
- 🦦 방법을 바꿔서 다시 시도한다

**Q3. 공부할 때 나는?**
- 🦊 빠르게 많이 보는 게 좋다
- 🐱 하나를 깊이 이해하는 게 좋다
- 🦦 결과물이 생기면 동기부여가 된다

### 배정 로직
```
단일 종족 2~3개 → 해당 종족
전부 다름       → Kenine (Pax 기본값)
나중에 설정에서 변경 가능
```

## 8. 앱 내 Pax 등장 시나리오

### 홈화면 일상 멘트 (14종 로테이션)
```
월 "이번 주 첫 날. 작게 시작해도 괜찮아."
화 "어제보다 조금만 더. 그게 전부야."
수 "중간이 제일 힘들어. 그래도 여기 있잖아."
목 "이번 주 거의 다 왔어. 오늘도 한 번만."
금 "오늘 끝내면 주말이 더 편해."
토 "쉬어도 돼. 근데 10분만 어때?"
일 "내일을 위해 오늘 하나만."
```

### 스트릭 시나리오
| 트리거 | Pax 상태 | 멘트 |
|--------|---------|------|
| 3일 | Happy | "3일 연속! 습관이 만들어지고 있어." |
| 7일 | Celebrate | "일주일! 이제 진짜 시작된 거야." |
| 30일 | Explorer 해금 | "30일. 너 이제 탐험가야." |
| 100일 | Guardian 해금 | "100일. Lexiox가 너를 기억할 거야." |
| 끊김 | Sleepy | "어제 못 왔구나. 괜찮아, 오늘부터 다시야." |

### 시험 기간 (Scholar Pax)
```
D-N일: "시험까지 D-{N}일. 오늘 집중할 파트 하나만 정하자."
당일:  "오늘이야. 지금까지 한 게 있잖아. 그거면 충분해."
이후:  "수고했어. 결과가 어떻든, 오늘 한 건 사라지지 않아."
```

### 레벨업 진화 연출
```
1. 화면 전체 에메랄드 빛
2. 현재 Pax 실루엣 → 빛 속으로 사라짐
3. 새 Pax 등장
4. 한 문장
   Cub→Explorer:   "이제 탐험 시작이야."
   Explorer→Scholar: "알면 알수록 더 알고 싶어지지?"
   Scholar→Guardian: "이제 혼자가 아니야."
   Guardian→Master:  "Lexiox의 빛이 네 안에 있어."
```

### 장기 미접속 복귀
```
7일:  Pax Thinking  "어디 갔었어? 기다렸어."
30일: 눈 뜨는 애니  "오랜만이다. 그냥 오늘 하나만."
복귀 첫날: 난이도 낮춤 + Pax 계속 등장
```

### Shattered Pax 등장 조건
```
일반 플로우에서는 등장 ❌
등장 가능:
  - 시즌 이벤트 스토리
  - 업적 해금 스토리컷
  - "Shattered를 구해라" 특별 챌린지
```
